git命令

2021-08-10 1 分钟阅读 #git#git
目录

git 提交

1.将当前已添加暂存区内容提交

git commit -m '<提交说明>'

2.将当前修改内容添加并提交

git commit -am '<提交说明>'

3.将当前修改内容追加到上一次提交记录中

git commit -a --amond -m '<提交说明>'

git 分支

  1. 创建分支
git branch <分支名称>
  1. 切换分支
git checkout <分支名称>
  1. 切换分支,不存在此分支时进行创建分支并切换
git checkout -b <分支名称>
  1. 合并分支(将其他分支合并到当前分支)
Merge合并会生成一条commit记录,并且分支状态为并行状态

git merge <分支名称>
  1. 合并分支(将当前分支合并到指定分支)
rebase 合并不会生成多余的commit记录,并且分支状态为线性状态

git rebase <分支名称>
